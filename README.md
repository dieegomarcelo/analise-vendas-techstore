# 📊 Análise de Vendas - TechStore

Projeto desenvolvido como parte de um **desafio da Rocketseat**, com foco em **análise exploratória de dados** da empresa fictícia **TechStore**.

## 🔎 Objetivos
- Explorar o dataset `vendas.csv`.
- Calcular receita total.
- Filtrar vendas da categoria **Eletrônicos**.
- Identificar o **produto mais vendido**.
- Descobrir a **região com maior valor de compras**.
- Criar visualizações simples.

## 🛠️ Habilidades aplicadas
- **Python** para análise de dados.
- **Pandas** para manipulação de tabelas.
- **Matplotlib** para gráficos.
- **EDA (Exploratory Data Analysis)**.
- **Git/GitHub** para versionamento e publicação.
